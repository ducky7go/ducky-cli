# 🔥 傷を暖かさに

ゲーム内でダメージを受けた際、このModは自動的に寒冷・凍傷状態を解除し、戦闘中も温かく快適に保ちます。

## ✨ 主な機能

- **自動的なネガティブ効果解除**: ダメージを受けると自動的に寒冷・凍傷バフを除去
- **即時発動**: ダメージを受けた瞬間に効果を発揮、手動操作不要
- **戦闘状態維持**: 寒冷効果による戦闘パフォーマンス低下を防止
- **軽量設計**: 最小限のリソース使用、ゲーム性能に影響なし

## 🎮 使用方法

Modは自動的に動作し、設定は不要です。敵や環境からダメージを受けると、寒冷・凍傷状態が即座に解除されます。

## ⚠️ 注意事項

- ダメージを受けた時のみ発動し、寒冷状態を能動的に予防しません
- 他のModと互換性があり、競合しません
- シングルプレイヤー・マルチプレイヤーの両方で動作します

## 🔧 技術詳細

- 全てのダメージタイプを検出（敵の攻撃、環境ダメージ、落下ダメージなど）
- 寒冷関連バフのスマート認識により、状態を正確に解除
- 効率的なイベントリスニング機構で迅速な応答を保証
