# 🔥 傷害轉溫暖

當你在遊戲中受到傷害時，這個模組會自動清除你身上的寒冷和凍傷狀態，讓你在戰鬥中保持溫暖舒適。

## ✨ 主要功能

- **自動清除負面狀態**：受到任何傷害時自動移除寒冷和凍傷buff
- **即時生效**：無需手動操作，受傷時立即生效
- **保持戰鬥狀態**：避免因寒冷狀態影響戰鬥表現
- **輕量級設計**：佔用資源極少，不影響遊戲性能

## 🎮 使用說明

模組會自動運行，無需任何設置。當你受到敵人傷害或環境傷害時，身上的寒冷和凍傷狀態會被立即清除。

## ⚠️ 注意事項

- 只在受到傷害時觸發，不會主動預防寒冷狀態
- 與其他模組兼容，不會產生衝突
- 適用於單人遊戲和多人遊戲

## 🔧 技術詳情

- 檢測所有類型的傷害（包括敵人攻擊、環境傷害、摔落傷害等）
- 智慧識別寒冷相關buff，精準清除目標狀態
- 高效的事件監聽機制，確保響應迅速
