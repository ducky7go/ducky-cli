# 🌡️ 被打红温

当你在游戏中受到伤害时，这个模组会自动清除你身上的寒冷和冻伤状态，让你在战斗中保持温暖舒适。

## ✨ 主要功能

- **自动清除负面状态**：受到任何伤害时自动移除寒冷和冻伤buff
- **即时生效**：无需手动操作，受伤时立即生效
- **保持战斗状态**：避免因寒冷状态影响战斗表现
- **轻量级设计**：占用资源极少，不影响游戏性能

## 🎮 使用说明

模组会自动运行，无需任何设置。当你受到敌人伤害或环境伤害时，身上的寒冷和冻伤状态会被立即清除。

## ⚠️ 注意事项

- 只在受到伤害时触发，不会主动预防寒冷状态
- 与其他模组兼容，不会产生冲突
- 适用于单人游戏和多人游戏

## 🔧 技术详情

- 检测所有类型的伤害（包括敌人攻击、环境伤害、摔落伤害等）
- 智能识别寒冷相关buff，精准清除目标状态
- 高效的事件监听机制，确保响应迅速
